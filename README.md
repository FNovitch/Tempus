<h3 align="center">Resumo do Projeto</h3>
<h6>
  Um site para consultar o clima e a velocidade do vento em uma cidade. O
  projeto foi feito com foco em responsividade, manipulacao do DOM, TypeScript
  no front-end e, agora, um back-end Node.js proprio para proteger a chave da
  API e centralizar a logica de consulta do clima.
</h6>

<h5>Tecnicas e tecnologias utilizadas</h5>

- `HTML`
- `CSS`
- `TypeScript`
- `Node.js`
- `API HTTP`
- `Integracao com API externa`
- `Cache em memoria`
- `Rate limit em memoria`

<h5>Como executar localmente</h5>

1. Crie um arquivo `.env` na raiz do projeto.
2. Adicione sua chave com `OPENWEATHER_API_KEY=sua_chave_aqui`.
3. Se quiser, ajuste a porta com `PORT=3000`.
4. Inicie o servidor com `node server/server.js`.
5. Abra `http://localhost:3000` no navegador.
